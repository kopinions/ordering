get /products

As a customer I can get all the products then I can get products to buy.

  • Request
  • Response

HTTP status code 200

Body

Type: application/json

Example: 
[
  {
    "uri": "/products/1",
    "name": "apple",
    "description": "red apple",
    "price": 1.1
  }
]

post /products

As a admin I can create a product so that customer can buy the product

  • Request
  • Response

Body

Type: application/x-www-form-urlencoded

Form Parameters
  • name (string)
  • description (string)
  • price (number)

HTTP status code 201
